redis 数据结构 set

2021-04-07 3 min read

增加元素
可以增加若干个元素的set 集合

 SADD key member [member ...]
  summary: Add one or more members to a set
  since: 1.0.0

获取集合中元素的数量

  SCARD key
  summary: Get the number of members in a set
  since: 1.0.0

获取差集
对比多个集合的差集，返回左侧集合中，其他集合中没有的元素 有点像是
(A-A∩B...C) 取集合A中和其他集合相交部分的元素后的差集

sdiff key [key ...]
summary: Subtract multiple sets
since: 1.0.0

获取差集，并且保存到一个新的集合中（主要还是A集合为主）

  sdiffstore destination key [key ...]
  summary: Subtract multiple sets and store the resulting set in a key
  since: 1.0.0

获取多个集合交集部分的元素

sinter key [key ...]
summary: Intersect multiple sets
since: 1.0.0

获取多个集合交易部分的元素并且保存到一个新的集合中

sinterstore destination key [key ...]
summary: Intersect multiple sets and store the resulting set in a key
since: 1.0.0

查看集合中是否包含该元素

  SISMEMBER key member
  summary: Determine if a given value is a member of a set
  since: 1.0.0

查看集合中元素

 smembers key
 summary: Get all the members in a set
 since: 1.0.0

将元素从原来的集合中转移到 目标集合中

  smove source destination member
  summary: Move a member from one set to another
  since: 1.0.0

随机弹出 count 个元素

spop key [count]
summary: Remove and return one or multiple random members from a set
since: 1.0.0

随机获取 count 数量个元素

srandmenber key [count]
summary: Get one or multiple random members from a set
since: 1.0.0

从集合中删除 一个或者多个指定元素

  srem key member [member ...]
  summary: Remove one or more members from a set
  since: 1.0.0

sscan 返回匹配 pattern 的 count 数量的元素

只有两种游标是合法的：

在开始一个新的迭代时， 游标必须为 0 。
增量式迭代命令在执行之后返回的， 用于延续（continue）迭代过程的游标。

 sscan key cursor [MATCH pattern] [COUNT count]
 summary: Incrementally iterate Set elements
 since: 2.8.0
 sscan myset0 0 match  n*

取多个集合的并集，重复的会被去重

  sunion key [key ...]
  summary: Add multiple sets
  since: 1.0.0

取多个集合的并集，去重后，保存为一个新的集合

  sunionstore destination key [key ...]
  summary: Add multiple sets and store the resulting set in a key
  since: 1.0.0
