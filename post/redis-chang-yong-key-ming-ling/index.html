redis 常用key命令

2021-04-12 2 min read

查看所有key
keys pattern

查看key的类型
type key

随即返回一个key
randomkey

删除key
del key [key2 key3 key4 ...]

迭代指定pattern 的key
返回 key 的列表：

scan cursor [MATCH pattern]  [COUNT count]

scan 0 match * count 1 
扫描 游标根据上次返回的 结果 ，0 表示一次遍历开始
最后以 0 返回表示，扫描一遍结束

判断key是否存在

返回状态1/0，True/False，当传入多个key时返回or的结果，即只要有一个存在就返回True：

exists key [key ...]

序列化

dump key

将 key 从当前db移动到另一个db中
返回1/0 ，true/false

move key db_index

重命名key

直接修改key的名称：

rename key newkey

过期策略

给指定的 key 设置过期时间，以秒或unix timestamp为单位：

    expire key seconds
    expireat key timestamp

给指定的 key 设置过期时间，以毫秒或unix timestamp为单位：

    pexpire key milliseconds
    pexpireat key milliseconds-timestamp 
    ttl key  查看 key 过期时间

移除 key 的过期时间，key 将持久保持：

persist key

以秒为单位，返回给定 key 的剩余生存时间(TTL, time to live)：

ttl key

以毫秒为单位返回 key 的剩余生存时间：

pttl key
